Frontend Development

Frontend Development is het laatste codeer vak wat ik heb gehad, dit borduurde voort op de eerdere vakken die om code draaide. Bij Frontend word van je verwacht dat je alle codetalen al goed onder de knie hebt en dat je klaar bent voor de volgende stap. Daarom word er bij Frontend gelet op de toegankelijkheid voor mensen met een beperking.

Bij frontend kies je een bestaande website en is het jouw taak om deze te verbeteren. Daarnaast heb ik ook de volledige site responsive gemaakt, hij is dus zowel op mobiel als computer volledig werkend!
